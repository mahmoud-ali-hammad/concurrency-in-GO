# Concurrency in Go

This repository is an explanation and companion guide for the book _Concurrency in Go_. It is organized into 6 chapters that follow the book’s structure and help readers understand the main ideas, patterns, and practical examples more clearly.

## About this repo

The goal of this repository is to:

- explain the concepts covered in the book
- provide a clean chapter-based structure
- make it easier to review concurrency topics in Go
- collect notes, examples, and summaries in one place
